File Description:
Actual_text             ¨C Actual text script
Actual_Notations        ¨C Original phonetic annotations
Suppose_text            ¨C Script used for Suppose
Suppose_Notations       ¨C Phonetic annotations after Suppose

Actual   ¨C The sentence or pronunciation that the speaker actually produced during the recording.
Supposed ¨C The sentence or pronunciation that the speaker was expected to produce (i.e., the target reference).

Annotation Tags:
<SPK/>  ¨C Speaker vocal noises such as laughter, cough, clearing throat, etc.
<NON/>  ¨C Non-speech vocalizations like breathing, sighs, swallowing, etc.
<NPS/>  ¨C Other miscellaneous vocalizations
<STA/>  ¨C Audio interference (e.g., overlapping speech, background noise, whistles, wind, rain, car noise)
**     ¨C Unclear or unintelligible words or syllables

Audio File Description:
Audio file names are numerical. For example:
0         ¨C Session number  
0010      ¨C Speaker ID  
0001      ¨C Sentence ID
